# Лабораторная работа "FrontEnd"

## Создание интерактивной веб-страницы с использованием HTML, CSS и JavaScript

### Цель работы
Освоить основы фронтенд-разработки через создание интерактивной веб-страницы, применяя комплексный подход к использованию HTML для структуры, CSS для стилизации и анимации, и JavaScript для реализации интерактивных функций.

### Задачи лабораторной работы
1) **Изучить и применить на практике** основные технологии фронтенд-разработки: HTML5, CSS и JavaScript
2) **Освоить техники CSS** для позиционирования элементов, создания визуальных эффектов и реализации анимаций
3) **Реализовать интерактивность** с помощью JavaScript
4) **Отработать навыки** отладки и тестирования веб-страницы в браузере
